## Tutorial Ngerun

1. Clone dulu ke pc anda pake `git clone https://github.com/MachsunSSR/Pra-Intern-KampusIndonesia.git`
2. `cd Pra-Intern-KampusIndonesia`
3. `npm install`
4. `npm run start`
5. Tungguin loadingnya & Selamat menikmati
